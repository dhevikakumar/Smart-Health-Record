MedicalSHR is a secure digital health record management system designed to ensure safe and controlled access to patient information.
It includes user authentication, role-based access control, emergency access workflows with approval mechanisms, QR code generation
for quick data retrieval, and AI-powered text summarization for medical records.

Tech Stack:
The frontend is built using React 19, Vite, Tailwind CSS, React Router, and Appwrite.
The backend is developed with Node.js and Express, integrated with Appwrite for backend services,
Twilio for SMS notifications, and OpenAI for generating intelligent text summaries.
